# PHP Login-System

Dieses Projekt ist ein einfaches Login- und Registrierungssystem,
das ich erstellt habe, um die Grundlagen von PHP, MySQL und Sessions
praktisch zu verstehen.

## Funktionen
- Benutzerregistrierung mit Passwort-Hashing
- Login-Funktion
- Session-basierter Zugriff auf geschützte Seiten

## Verwendete Technologien
- PHP
- MySQL
- HTML (Formulare)

## Hinweis
Das Projekt ist lokal mit XAMPP umgesetzt.
Da GitHub kein PHP ausführt, dient dieses Repository
zur Einsicht in den Code und die Projektstruktur.
